Introduction¶

K-Means Clustering is one of the popular unsupervised machine learning algorithms. Generally, clusters or groups are asked to find when we don’t have labeled data. This algorithm classify the data without being trained with labeled data. Once the whole data is classified in different clusters, any new data can be assigned to the most relevant one.

On the other hand, Random forests is a supervised learning algorithm. It creates decision trees on randomly selected data samples, gets prediction from each tree and selects the best solution by means of voting. This method can also calculate the importance of different features in predicting. We can build the model based on important features only and then test the model.

Problem Definition¶

Given Iris data set from scikit learn. We apply K-means clustering to classify the data. This dataset consists of three species of Iris (Iris setosa, Iris virginica and Iris versicolor) each having 50 samples. The number of feature variables is four: length and width of sepals and petals.

At first, we assume that we do not know the different species of the Iris flowers and apply K-means algorithm to classify the whole dataset into different clusters. Our goal is to observe, whether we get the cluster no as 3 or not.

Next, we first label the data and then will split the whole data into train and test set. We will apply Random forests algorithm to train the model using the train dataset and then will apply to the test set. Finally we calculate the confusion matrix to understand the quality of the fiting model. Another, important part is to reduce the number of features based on their importance in prediction. We then rebuild the model and want to compare accuracy for both the cases.

Finding Optimum number of cluster (Elbow Method)¶

To find the optimum nuber of cluster for classification, we will now apply Elbow method on this Iris dataset.

In [28]:
#Finding the optimum number of clusters for k-means classification
from sklearn.cluster import KMeans
Sum_of_Inertias = []

for i in range(1, 12):
    kmeans = KMeans(n_clusters = i, init = 'k-means++', max_iter = 300, n_init = 10, random_state = 1)
    kmeans.fit(iris.data)
    Sum_of_Inertias.append(kmeans.inertia_)
In [29]:
#Plotting the results onto a line graph, allowing us to observe 'The elbow'
plt.plot(range(1, 12), Sum_of_Inertias,'o-')
plt.title('The elbow method')
plt.xlabel('Number of clusters')
plt.ylabel('Sum of Inertias') #within cluster sum of squares
plt.show()

Comment: The Elbow method shows that the optimum no of clusters we can choose is 3.

Finding Important Features in the model¶

Usually, when a dataset has many number of features, it is important to find out which are important. Random forest helps us find relative importance or contribution of each feature in the prediction. It computes the relevance score of each feature in the training phase.

This will help us to select important features and reject the other features which are less important.

We use the feature importance variable to see feature importance scores.
