Introduction¶

Linear regression is one type of supervised learning model where there is a linear relationship between the independent and dependent variable. It predicts a continuous response depending on the history of the initial data. Here, we use linear regression on a given Advertisement dataset to study how the money spent on advertisement impacts sales.

Problem Definition¶

Given a dataset which contains the amount of money spent on advertisement and the sales of a widget. By analyzing the dataset, we want to address how one (any company which manufactures this widget) should spend advertising money in the future?

Interpretation:¶

As we observed, the confidence interval for Newspaper includes zero only, others don't. The p-values for TV and radio are approximately zero, where for Newspaper it is approximately 0.86.

The conventional Null Hypothesis is:

  There is no relationship between the features and the response.

Alternative Hypothesis:

  There is a relationship between the features and the response.

From the above 95% Confidence interval data and p-values, we note the following:

           1. For TV and Radio ads, the Confidence interval does not include zero and p-value is approximately zero (<0.05)
           2. For Newspaper ads, the Confidence interval include zero and the p-value is approximately 0.86.

Based on these, we can say that we reject the null hypothesis for the TV and Radio add, i.e., there is a linear dependence of the TV and Radio ads on the Sales. And, for Newspapers, we fail to reject the null hypothesis. So We go for the alternative hypothesis.

Categorical Features with Two Categories¶¶

So far what we have studied the datasets where all the features are numeric. Sometimes few features may be categorical too. In such a case, how would we apply Linear Regression technique?

For example, we create a new feature called Size, and randomly assign observations to be small or large:

In [52]:
# set a seed for reproducibility
np.random.seed(58411)

# For two categories, we can assign each of them with an equal probability (0.5)
# create a Series of booleans in which roughly half are True
nums = np.random.rand(len(data))
mask_large = nums > 0.5

# initially set Size to small, then change roughly half to be large
data['Size'] = 'small'

# Series.loc is a purely label-location based indexer for selection by label
data.loc[mask_large, 'Size'] = 'large'
data.head()

Now we can't work with this categorical data in scikit-learn, we need to map it to numerical values. The feature having two categories only, we use binary dummy variable to represent it : small=0 and large=1.

Here, we study one example of how to deal with categorial feature with two levels only. That's why we created one dummy binary variable. We can extend the idea to a categorical feature with any no of levels. In general, for a categorical feature with k "levels", we need to create k-1 dummy variables.
